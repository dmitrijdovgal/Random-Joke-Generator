😂 Random Joke Generator – Multi‑Language Edition

A fun **joke generator** that serves up random jokes from a built‑in collection, with features to add, remove, categorize, save, and load jokes.  
Perfect for lightening the mood, learning programming basics, or integrating into your projects.  
Built in **7 programming languages** – each implementation shares the same core functionality.

## ✨ Features
- **Random joke** – get a random joke from the current collection.
- **Joke list** – view all jokes (with categories).
- **Add joke** – add a new joke with an optional category.
- **Remove joke** – delete a joke by its ID.
- **Categories** – filter jokes by category or see all categories.
- **Save/Load** – persist jokes to a JSON or text file.
- **Interactive CLI** – intuitive menu with numbered options.

## 🚀 How to Run
Each file is standalone – run it with the appropriate interpreter/compiler.

| Language | Command |
|----------|---------|
| Python   | `python joke_generator.py` |
| Go       | `go run joke_generator.go` |
| JavaScript | `node joke_generator.js` |
| C#       | `dotnet run` (or `csc JokeGenerator.cs && JokeGenerator.exe`) |
| Java     | `javac JokeGenerator.java && java JokeGenerator` |
| Ruby     | `ruby joke_generator.rb` |
| Swift    | `swift joke_generator.swift` |

## 🔧 Command‑line Arguments (Optional)
Most implementations also accept:
- `--joke` – show a single random joke and exit.
- `--add "joke text" --category "cat"` – add a joke from the command line.
- `--file jokes.json` – load jokes from a file on start.
